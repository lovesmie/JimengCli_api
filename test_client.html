开发者 API 集成文档

如何获取 API Key

  1. 联系管理员，进入后台控制台「API 令牌分发」页面
  2. 管理员填写拥有者标识并点击「签发令牌」
  3. 令牌签发后明文显示一次，需立即复制保存
  4. 此后令牌脱敏展示，若丢失需删除后重新签发

全局规范声明

Base URL：http://<server-ip>:3000/v1

认证 Header：Authorization: Bearer sk-jm-xxxxxxxxxx

请求格式：multipart/form-data — 所有生成接口统一使用 form-data，支持文件上传

任务轮询：GET /v1/tasks/:id — 轮询直到 status 为 success 或 failed

异步任务响应机制 (必读)

两个生成接口均为异步非阻塞：POST 提交后立即返回任务 ID，后台持续执行。客户端需轮询 GET /v1/tasks/:id 取得最终 URL。

① POST 提交 → 立即返回

{
  "id": "clxxxxxxxxxxx",
  "status": "processing",
  "submit_id": "abc123"
}

id 用于后续轮询，submit_id 为 CLI 流水号

② 轮询中 (每 3-5s)

{
  "id": "clxxxxxxxxxxx",
  "status": "processing"
}

继续轮询，最长等待约 5 分钟

③a 成功

{
  "id": "clxxx",
  "status": "success",
  "data": [{ "url": "https://cdn.xxx/out.jpg" }]
}

③b 失败

{
  "id": "clxxx",
  "status": "failed",
  "error": "错误原因说明"
}

接口1：图片生成 (POST /v1/images/generations)

POST {{ config.baseUrl.replace(/\/$/, '') }}/images/generations

📊 各模型参数约束

model resolution_type ratio 图生图 (images) 备注
5.04k / 2k全部比例✅ 最多 10 张旗舰推荐
4.64k / 2k全部比例✅ 最多 10 张高质量
4.54k / 2k全部比例✅ 最多 10 张
4.14k / 2k全部比例✅ 最多 10 张
4.04k / 2k全部比例✅ 最多 10 张
3.12k / 1k全部比例❌ 不支持旧版，降级分辨率
3.02k / 1k全部比例❌ 不支持旧版基础

📝 Form-Data 参数

  • prompt
    *必填 画面描述提示词
  • model
    模型版本: 5.0, 4.6, 4.5, 4.1, 4.0, 3.1, 3.0
  • ratio
    画幅比例: 16:9, 1:1, 9:16, 4:3, 3:4, 21:9 等
  • resolution_type
    分辨率: 4k / 2k (3.x 系最高 2k)
  • images
    [File Array] 图生图: 上传 1~10 张，仅限 4.0+ 模型

cURL 示例

curl -X POST {{ config.baseUrl.replace(/\/$/, '') }}/images/generations \
  -H "Authorization: Bearer YOUR_KEY" \
  -F "model=5.0" \
  -F "prompt=一只超写实机械猫咪" \
  -F "ratio=16:9" \
  -F "resolution_type=4k"

# 图生图 (4.0+ 限定):
# 追加 -F "images=@/path/to/ref.jpg"

POST {{ config.baseUrl.replace(/\/$/, '') }}/videos/generations

📊 各模型参数约束

model 时长 (s) 最高分辨率 文生视频 图生视频 多模态/音频 备注
seedance2.0_vip4–15720p✅✅✅ 全支持VIP 超极速
seedance2.0fast_vip4–15720p✅✅✅ 全支持VIP 极速版
seedance2.04–15720p✅✅✅ 全支持标准推荐
seedance2.0fast4–15720p✅✅✅ 全支持快速出图
3.5pro4–121080p✅✅❌高清，无多模态
3.0pro3–101080p✅✅❌
3.0fast3–10720p✅✅❌快速版
3.03–10720p✅✅❌旧基础版

📝 Form-Data 参数

  • prompt
    *必填 核心描述提示词
  • model
    模型名称，见上表
  • duration
    时长(整数秒)，范围见上表
  • ratio
    画幅比: 16:9, 9:16, 1:1, 4:3 等
多模态文件槽
  • image — 1张=图生视频；≥2张=触发多模态 (限seedance2.0系)
  • video — 0~3个视频文件，存在则触发多模态
  • audio — 0~3首音频，4~15秒，存在则触发多模态

cURL 示例

# 文生视频
curl -X POST {{ config.baseUrl.replace(/\/$/, '') }}/videos/generations \
  -H "Authorization: Bearer YOUR_KEY" \
  -F "model=seedance2.0" \
  -F "prompt=赛博朋克都市夜景" \
  -F "duration=5" -F "ratio=16:9"

# 图生视频 (单图定帧)
curl ... -F "image=@face.jpg"

# 多模态 (多图+音频)
curl ... -F "image=@face1.jpg" \
  -F "image=@face2.jpg" \
  -F "audio=@voice.mp3"
