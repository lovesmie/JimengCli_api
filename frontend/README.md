# 即梦 (Jimeng) API 管理后台

这是一个简单易用的管理后台界面，帮你把「即梦 CLI」无缝转换成一个通用的 OpenAI API 接口！通过这个页面，你可以像大管家一样轻松管理你的即梦账号和分配 API Key。

## 🌟 这个后台能帮你干什么？

1. **管理即梦账号（账号池系统）**
   - 你可以同时挂载多个即梦账号，系统在底层把它们全隔离开，绝对互不影响。
   - 点一下 **「🔑 扫码绑定」**，然后去浏览器或者手机扫一扫就能登录，非常傻瓜式！
   - 点一下 **「📡 连通测试 / 查余额」**，就能一键检查这个账号是不是掉线了，并且直接看它还剩下多少画图点数。

2. **生成万能的 API Key**
   - 点击 **「➕ 颁发并生成 Key」**，它会给你发一个长得像 `sk-jm-...` 的密钥。
   - 把哪怕一长串的密钥，填到你平时用的任何支持 OpenAI 格式的软件（比如 Chatbox、沉浸式翻译、或者你的其他小工具）里面，就能直接调用即梦的生图、生视频能力啦！

## 🚀 怎么跑起这个界面？

如果你是开发者或者刚拿到代码，只需要简单的两步走：

1. **安装必须要的包**：
   打开命令行（在 `frontend` 文件夹下），运行：
   ```bash
   npm install
   ```

2. **启动界面**：
   运行下面这个命令：
   ```bash
   npm run dev
   ```

3. **进入管理页面**：
   启动成功后，在浏览器里打开弹出的本地网址（通常是 `http://localhost:5173`），就能看到漂漂亮亮的管理后台啦！
